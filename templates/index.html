Jak na to
  1. 1 Nahrajte ceník CPC — XLSX s kategoriemi a cenovými hladinami. Lze nahrát více ceníků pro porovnání v čase.
  2. 2 Tabulka se zobrazí hned. Každý sloupec = jeden ceník, v každé buňce vidíte CPC a změnu oproti předchozímu.
  3. 3 Volitelně načtěte Heureka API — přidá reálná data (prokliky, % nákladů, % tržeb) pro každou kategorii.
Formát ceníku: první sloupec = ID kategorie, druhý = název, další sloupce = cenové hladiny (0.3, 0.5 …). Název souboru se použije jako datum/label.
